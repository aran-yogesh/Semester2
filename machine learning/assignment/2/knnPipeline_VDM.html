Problem: Using pipeline for KNN¶

In this problem you apply built-in KNN package to the healthcare data to predict adherent class (pdc-80-flag). Use of pipeline is recommended. Predict the pdc-80-flag using the following features ”pre-rx- cost”,”numofgen”,”numofbrand”,”generic-cost”,”adjust-total-30d”, and ”num- er”. Determine the accuracy rate for test set for k = 75 to 105 with a step size of 2 and report it in a table. Use linear normalization method to normalize the input features and Euclidean distance for distance measure. Note that you must use the training parameters for normalization of test points

def knn_model(k, X_train, y_train, X_test, y_test): 
    model = Pipeline(
        steps = [
            ('preprocessor', columns_preprocessor),
            ( 'KNN', KNeighborsClassifier(n_neighbors = k)),
        ]
    )
    model.fit(X_train, y_train)
    y_test_pred = model.predict(X_test)
    accuracy = accuracy_score(y_test, y_test_pred)
    return(accuracy)
